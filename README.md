
🤖 Customer Subscription Prediction (Decision Tree Classifier)
📌 Project Overview

This project focuses on building a machine learning model to predict whether a customer will subscribe to a service. Using a Decision Tree Classifier, the goal was not only to achieve good accuracy but also to understand how the model makes decisions.

🎯 Objectives

Predict customer subscription (Yes/No)

Identify key factors influencing customer decisions

Interpret model behavior and feature importance

Apply insights to real-world business scenarios

🧠 Model Used

Decision Tree Classifier

Chosen for its interpretability and ease of understanding decision logic

📊 Key Insights
🔍 Feature Importance

Call Duration contributes over 60% to predictions

Previous Positive Responses significantly increase subscription likelihood

Customers with housing loans are less likely to subscribe

🧪 Real-World Prediction

Tested on a new customer profile

Model predicted:

Probability: 34.7%

Outcome: ❌ Not Subscribed

📈 Model Performance

Accuracy: ~79.5%

Balanced performance with interpretable results

💡 Key Takeaway

Understanding why a model makes decisions is as important as the prediction itself.
This helps businesses make data-driven and transparent decisions.
